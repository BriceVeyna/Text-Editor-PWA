# Text-Editor-PWA
This application is a browser-based text editor that can be used offline.

## Description
This project takes a base text editor starter code and turns it into a progressive web application that can be installed and used offline.

### Deployed Link
[Text-Editor-PWA](https://text-editor-pwa-01.herokuapp.com/)
